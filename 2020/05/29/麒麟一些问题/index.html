    麒麟一些问题

    主要是麒麟遇到的一些问题

    1. 解决光盘直接挂载不能使用源的问题

    echo "deb file:///media/kylin/kylin-os-4.0.2 juniper main restricted universe multiverse" >> /etc/apt/sources.list

apt-get --allow-unauthenticated update
apt-get --allow-unauthenticated install -y mysql-server

    如需永久化则

    echo "deb [trusted=yes] file:///media/kylin/kylin-os-4.0.2 juniper main restricted universe multiverse" >> /etc/apt/sources.list

    2. sp3系统不能执行脚本

    由于内部一些安全机制，导致不能直接在笔记本上执行自行编写的脚本文件。可通过一下命令放行。

    sudo setstatus softmode -p
